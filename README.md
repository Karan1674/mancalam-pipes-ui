## 📌 Overview

Mancalam Pipes is a modern, responsive product showcase interface designed to present industrial products in a clean, structured, and interactive way.

The project focuses on:
- High-quality UI/UX design
- Modular and maintainable code structure

- Smooth, intuitive user interactions
- Performance-friendly implementation using core web technologies

This project demonstrates how powerful and scalable applications can be built using **pure HTML, CSS, and JavaScript**, without relying on frameworks.

---

## 🛠️ Tech Stack

- HTML5 (Semantic Structure)
- CSS3 (Flexbox, Grid, Modular Styling)
- JavaScript ES6 (DOM Manipulation, Events)
- Font Awesome (Icons)
- Google Fonts (Inter, Urbanist)

---

## 🎯 Core Features

### 🔹 1. Fully Responsive Layout
- Optimized for desktop, tablet, and mobile
- Fluid containers and flexible grids
- Adaptive typography and spacing

---

### 🔹 2. Scroll-Based Sticky Header
- Appears only after crossing first viewport (first fold)
- Disappears when returning to top
- Smooth transitions with CSS animations
- Enhances navigation without clutter

---

### 🔹 3. Interactive Image Carousel
- Thumbnail-driven image switching
- Previous / Next navigation
- Active state management
- Seamless transitions between images

---

### 🔹 4. Advanced Image Zoom (Lens Effect)
- Hover-based zoom interaction
- Dynamic lens positioning
- Real-time magnified preview
- Smooth tracking of cursor movement

---

### 🔹 5. Horizontal Application Slider
- Scrollable card layout
- Mouse wheel → horizontal scroll behavior
- Navigation buttons with dynamic enable/disable states
- Smooth scrolling experience

---

### 🔹 6. FAQ Accordion System
- Expand/collapse interaction
- Only one active item at a time
- Clean UX behavior

---

### 🔹 7. Reusable Modal System
- Multiple modal triggers supported
- Overlay click to close
- Scroll lock when modal is active
- Form interaction handling

---

### 🔹 8. Process Tabs (Step Navigation)
- Tab-based interaction system
- Mobile-friendly next/prev navigation
- Dynamic active state management

---

### 🔹 9. Testimonials Slider
- Horizontal scrolling layout
- Mouse wheel interaction
- Smooth UX without heavy libraries

---

### 🔹 10. Dynamic Footer
- Automatically updates current year using JavaScript
- Eliminates need for manual updates
